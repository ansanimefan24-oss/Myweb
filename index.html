كالي لينكس

دروس وأدوات ونصائح احترافية

تنصيب كالي

شرح كامل لتثبيت كالي لينكس على VirtualBox أو على جهاز حقيقي، مع إعداد الشبكة وتحديث النظام.

أدوات مهمة

تعلم استخدام Nmap, Wireshark, Metasploit، وكيفية قراءة نتائج الفحص وتحليلها.

اختراق أخلاقي

خطوات اختبار الاختراق الأخلاقي بطريقة قانونية: التخطيط، الفحص، الاستغلال، والتقرير النهائي.

البرمجة

من الصفر حتى الاحتراف

Python

أساسيات Python، مشاريع عملية، وأمثلة على تطبيقات الأتمتة والـ scripting في كالي.

JavaScript

تعلم JS من الصفر، DOM، AJAX، وكتابة أدوات صغيرة تساعد في الاختراق الأخلاقي.

HTML + CSS

بناء واجهات احترافية، تصميم صفحات ثابتة، وتجهيز موقعك الخاص لتقديم محتوى احترافي.

مشروع 1: سكريبت فحص الشبكات
